# 프리온보딩 지원자 임수찬

이 프로젝트는 created-react-app으로 제작되었습니다.

## 화면 구성

크게 프로필(홈화면), 로그인, 회원가입, 투두리스트 네개의 화면으로 구성이 되어있습니다.

### `npm start`

올려드린 깃헙주소를 vscode로 여신 후, 'npm start'를 입력하시고 Enter키를 누르시면 정상적으로 프로젝트를 실행하실 수 있습니다.
